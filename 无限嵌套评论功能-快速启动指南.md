## 🔍 常见问题

### Q1: 后端启动失败，提示端口被占用？
**解决方案**:
```powershell
# 查找占用8000端口的进程
netstat -ano | findstr :8000

# 终止进程（替换<PID>为实际进程ID）
taskkill /PID <PID> /F
```

### Q2: 前端启动失败，提示端口被占用？
**解决方案**:
```powershell
# 查找占用3002端口的进程
netstat -ano | findstr :3002

# 终止进程
taskkill /PID <PID> /F
```

### Q3: 评论发表后看不到？
**可能原因**:
1. 后端服务未启动
2. 数据库迁移未执行
3. 浏览器缓存问题

**解决方案**:
```powershell
# 重新执行数据库迁移
cd backend
Get-Content migrate_add_comment_parent_id.sql | mysql -u root -p123456 liususpace

# 刷新浏览器（Ctrl+F5 强制刷新）
```

### Q4: 点击回复没有反应？
**可能原因**:
1. 未登录
2. 组件未正确加载

**解决方案**:
1. 检查是否已登录（查看右上角是否有用户信息）
2. 打开浏览器控制台（F12），查看是否有错误信息
3. 刷新页面重试

### Q5: 嵌套层级太深，显示混乱？
**建议**:
- 当前实现支持无限嵌套
- 如果层级过深（>5层），建议限制最大显示层级
- 可以在 `CommentItem.vue` 中添加层级判断

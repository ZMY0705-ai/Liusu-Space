# 我的讨论功能 - 实现说明

## 📋 功能概述

在个人中心的"我的"页面中新增"我的讨论"功能，用户可以查看自己在论坛发表的所有帖子。

## ✅ 已完成的工作

### 1. 后端接口（forum.py）

**新增接口**: `GET /forum/my-posts`

- **路径**: `/forum/my-posts`
- **方法**: GET
- **认证**: 需要 JWT Token
- **功能**: 获取当前登录用户的帖子列表
- **参数**:
  - `skip`: 偏移量，默认 0
  - `limit`: 每页数量，默认 100
- **返回**: 帖子列表（包含作者信息）

```python
@router.get("/my-posts", response_model=List[schemas.PostResponse])
def get_my_posts(
    current_user: models.User = Depends(get_current_user),
    skip: int = 0,
    limit: int = 100,
    db: Session = Depends(get_db)
):
    """获取当前用户的帖子列表"""
    from sqlalchemy.orm import joinedload
    posts = db.query(models.ForumPost).options(
        joinedload(models.ForumPost.author)
    ).filter(
        models.ForumPost.author_id == current_user.id
    ).offset(skip).limit(limit).all()
    return posts
```

### 2. 前端 API 封装（work.ts）

**新增函数**: `getMyPosts()`

```typescript
// 获取我的帖子列表
export function getMyPosts(skip: number = 0, limit: number = 20) {
  return http.get<any[]>('/forum/my-posts', {
    params: { skip, limit }
  })
}
```

   ```typescript

     try {
       const response = await getMyPosts(0, 100)
       myPosts.value = response.data || []

       console.error('加载讨论失败:', error)
     }
   }
   ```

## 🔍 技术要点

### 数据流
```
用户访问个人中心
  ↓
onMounted 触发
  ↓
loadMyContent() 调用
  ↓
并行加载：作品、草稿、收藏、讨论
  ↓
myPosts.value 更新
  ↓
Vue 响应式渲染列表
```

### 权限控制
- 接口需要 JWT 认证
- 未登录用户看不到"我的讨论"菜单
- 自动过滤只显示当前用户的帖子

### 性能优化
- 一次性加载所有帖子（limit=100）
- 后续可优化为分页加载
- 使用 keep-alive 缓存页面状态

## 🐛 常见问题

### Q1: 列表为空但数据库中有帖子？
**检查**:
1. 确认已登录且 token 有效
2. 检查帖子是否属于当前用户（author_id 匹配）
3. 查看浏览器控制台是否有 API 错误

### Q2: 点击帖子无法跳转？
**检查**:
1. 确认路由配置正确（`/forum/:id`）
2. 检查 ForumDetail.vue 页面是否存在
3. 查看 router.push 是否被正确调用

### Q3: 统计数据不准确？
**检查**:
1. 确认 `myPosts.length` 是否正确计算
2. 检查 `loadMyPosts()` 是否在 `loadMyContent()` 中被调用
3. 查看网络请求是否成功返回数据
